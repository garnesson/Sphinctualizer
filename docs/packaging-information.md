# Packaging Information

To package the skin and layouts open Rainmeter and press *Create .rmskin package...* and do the following:

- Information
	- **Name**: Sphinctualizer
	- **Author**: Gustav Blomér (gkj.se)
	- **Version**, enter the corresponding version number
- Components
	- Use *Add skin...* to add the *Sphinctualizer*-folder within the *Skin*-folder
	- Use *Add layouts...* to add each of the folders from within the *Layouts*-folder
- Press *Next*
- *Options*-tab
	- Change *Save package to* to the be located in the *Packaged*-folder
	- After installation
		- **Load skin**: Sphinctualizer\Medium.ini
	- Minimum requirements
		- **Rainmeter version**: 3.3.0
		- **Windows version**: 7
- *Advanced*-tab
	- **Variables files**: Sphinctualizer\@Resources\Variables.inc
- Press *Create package*
